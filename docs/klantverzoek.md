# klantverzoek

## Definitie
vraag van een klant om iets te doen of te laten

## Toelichting
Hoewel een klacht slechts een uiting is van ontevredenheid, beschouwen we deze als een klantverzoek omdat de netbeheerder hier binnen een bepaalde termijn opvolging aan moet geven.
